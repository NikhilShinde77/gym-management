
# Gym Management System

## Project Overview

The **Gym Management System** is a web-based application designed to automate and manage the daily operations of a gym. The system allows administrators, trainers, and members to interact through different panels to efficiently manage memberships, workout plans, attendance, and payments.

This project helps reduce manual work by digitizing gym management processes and improving communication between members, trainers, and administrators.

---

## Project Objectives

* Automate gym membership management.
* Track member attendance and subscription status.
* Manage trainer assignments and workout schedules.
* Monitor member performance and progress.
* Generate reports related to revenue and membership.

---

## System Users

### 1. Member Panel

Members can manage their gym activities through the following features:

* Register / Login to the system
* Choose and subscribe to membership plans
* View personal workout schedule
* Mark daily attendance
* Check subscription status
* View payment history

---

### 2. Trainer Panel

Trainers can manage training sessions and monitor member progress.

* Secure trainer login
* View assigned gym members
* Create workout plans for members
* Update member progress
* Manage training schedules

---

### 3. Admin Panel

The administrator controls the entire system.

* Manage members and trainers
* Create and update membership plans
* Monitor member attendance
* Generate revenue reports
* Maintain and manage system records

---

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Gym-Management-System.git
```

### 2. Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### 3. Run the Application

Start Backend

```bash
npm run server
```

Start Frontend

```bash
npm run dev
```
